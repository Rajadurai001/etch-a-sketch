# Etch-a-Sketch 🎨

A browser-based sketch pad built with **HTML, CSS, and JavaScript**.  
This project is part of a learning exercise to practice **DOM manipulation, Flexbox layout, and event handling**.

## Features
- Creates a default **16x16 grid** of square divs using JavaScript.
- Squares change color when hovered, leaving a pixelated trail like a pen.
- Button to reset the grid size:
  - Prompts the user for a new grid dimension (up to 100).
  - Generates a new grid in the same fixed space (960px wide).
- Responsive grid using **Flexbox** (no CSS Grid).
- Clean separation of HTML, CSS, and JavaScript files.

## Extra Credit ✨
- **Random colors:** Each hover generates a random RGB value.
- **Progressive darkening:** Each hover increases opacity by 10%, reaching full black after 10 passes.
